## 📱 **How to Test These Codes:**

### **Method 1: Manual Entry in App**
1. Open Opilex app → Dashboard
2. Tap "Scan Now" button (bottom center)
3. Select "Enter Manually" tab
4. Copy any JSON code from above
5. Paste into the text field
6. Tap "Verify Code"
7. Observe the result (success/error message)

### **Method 2: Expected Results**

#### **Codes 1-10 (Valid)**: ✅ Success
- Should show "Authentic Wire Verified!"
- Points should be added to balance
- Different point amounts (25-250 points)

#### **Codes 11-20 (Promotional)**: ✅ Success + Bonus
- Should show "Authentic Wire Verified!"
- Bonus points included in total
- Promotion details displayed

#### **Codes 21-30 (Invalid)**: ❌ Error Messages
- Code 21: "Already Used" error
- Code 22: "Expired Wire" error  
- Codes 23-30: "Invalid Wire" errors

### **🔍 Testing Tips:**
- Test codes in order to see different behaviors
- Check your points balance before and after
- Try both valid and invalid codes
- Note the different error messages
- Test promotional codes for bonus points
