## Image Requirements:
- **Width**: 350-400px (will be scaled to fit screen width)
- **Height**: As specified per banner type
- **Format**: JPG or PNG
- **Quality**: High resolution for crisp display

## Usage:
These images are loaded using React Native's `require()` function and displayed in the ImageSlider component.
